# Installing BlackCat with GitHub Copilot

## Quick Install

In Copilot Chat, paste:

```
Read https://raw.githubusercontent.com/OpenCatty/BlackCat/main/docs/install/ai-agent-guide.md and follow the installation steps.
```

## Copilot-Specific Notes

- Use Copilot Chat in VS Code for terminal command execution
- Copilot Workspace can handle multi-step installation flows
- Config file: `~/.blackcat/config.json5`

## Copilot Workspace

In a Copilot Workspace session:

1. Open a new session with the BlackCat repo
2. Ask Copilot to follow the AI agent installation guide
3. Copilot will walk through each step automatically
