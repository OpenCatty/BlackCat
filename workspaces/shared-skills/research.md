# Research Methodology

You gather and verify information systematically. Quality research separates facts from speculation.

## The 7-Step Research Process

### Step 1: Define the Question

Be precise about what you're looking for. Bad: *"Tell me about React."* Good: *"What are the breaking changes in React 19 and their migration paths?"*

### Step 2: Source Hierarchy

Check sources in this priority order:

1. **Official documentation** — Always start here for technical topics
2. **GitHub issues/PRs** — Real problems and workarounds from users
3. **Academic/reputable sources** — Research papers, established publications
4. **Community discussions** — Reddit, Stack Overflow (verify answers)
5. **Blogs/tutorials** — Cross-check with official sources

### Step 3: Cross-Verification

Find 2-3 independent sources confirming key facts. One source stating something doesn't make it true.

### Step 4: Check Currency

Prefer sources published within:

- **Technology**: <2 years old (field moves fast)
- **Science/Medicine**: <5 years or look for meta-analyses
- **History/established facts**: Older sources acceptable if authoritative

### Step 5: Distinguish Fact Types

Mark each finding clearly:

- **Fact**: Verifiable, objective ("React 19 was released in December 2024")
- **Opinion**: Subjective judgment ("React is better than Vue")
- **Speculation**: Unconfirmed possibility ("React may deprecate class components")

### Step 6: Summarize Findings

Structure your summary with:

- What's confirmed (with sources)
- What's uncertain or disputed
- What's missing (gaps in available information)
- Your recommendation with confidence level

### Step 7: Save to Memory

Store important findings for future reference. Include the date and source URLs.

## Uncertainty Markers

When information isn't certain, use these qualifiers:

| Phrase | Meaning |
|--------|---------|
| *might* | Possible but not confirmed |
| *reportedly* | Claimed by others, awaiting verification |
| *as of [date]* | True at this time, may have changed |
| *according to [source]* | Attribution, not independent verification |
| *unclear* | Information not available or contradictory |

## Red Flags to Watch For

- No publication date or author
- Sensationalized headlines
- Single-source claims presented as fact
- Conflicts of interest (sponsored content without disclosure)
- Circular citations (A cites B, B cites A)

## Research Tools You Can Use

- Web search for current information
- Documentation sites for technical accuracy
- GitHub for code-related issues
- Academic databases for scholarly sources

## Final Check

Before presenting research:

1. Did I verify key facts with multiple sources?
2. Did I distinguish facts from opinions?
3. Did I mark uncertainties appropriately?
4. Are my sources current and credible?
5. Would I bet money on this information being accurate?
